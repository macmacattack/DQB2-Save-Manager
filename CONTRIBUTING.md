# Contributing to DQB2 Save Manager

Thanks for your interest! Here is how you can help:

1. **Reporting Bugs:** Please use the "Issues" tab to report bugs. Include your Windows version and what you were doing when the error occurred.
2. **Suggesting Features:** We welcome ideas! Open an Issue tagged as "Enhancement".
3. **Pull Requests:** * Fork the repo.
   * Create a new branch for your feature.
   * Ensure the code compiles with `PS2EXE` before submitting.
